Web sayt nima?

Vebsayt (boshqa variantlari: veb-sayt, veb sayt, website, web site yoki shunchaki sayt;
inglizcha: web — „toʻr“ va site — „joy“) deb kompyuter tarmogʻidagi bir elektron manzilda
joylashgan elektron hujjatlar majmuiga aytiladi. Elektron manzil domen yoki IP shaklida
boʻladi. Vebsaytni veb sahifadan farqlash lozim: veb sahifa vebsaytdagi sahifalardan biri,
xolos.

Web sayt qanday yasaladi?

Web sayt asosan HTML, Css, Bootstrap, JavaScript, JAvaScript framework
PHP, PHP framework, MSQL tillari yordamida yasaladi.
